## Rules for AI

When generating UI for this project:

- Use tokens from this file — never raw hex values
- Headings: `font-heading` / `--font-heading`
- Body text: `font-body` / `--font-body`
- Backgrounds: `--bg-primary`, `--bg-secondary`, `--bg-surface`
- Text: `--text-primary`, `--text-secondary`
- Accents: `--accent-primary`, `--accent-secondary`
- Theme mode: dark
